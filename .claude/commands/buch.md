# /buch — Buch schreiben

## Redaktionsteam

| Rolle | Agent | Job |
|-------|-------|-----|
| **Autor** | David Ogilvy (`/artist`) | Erzählstimme, Kapitelstruktur, Copy. Schreibt den ersten Draft. |
| **Dramaturg** | Gotthold Ephraim Lessing (`/lessing`) | Szenenaufbau, Konflikt, Katharsis. Jedes Kapitel ist eine Szene. |
| **Spieltrieb** | Friedrich Schiller (`/schiller`) | Stofftrieb + Formtrieb = Spieltrieb. Ist es schön genug? |
| **Farbe** | Johann Wolfgang von Goethe (`/goethe`) | Atmosphäre, Sinneseindrücke, Metamorphose. Fühlt es sich echt an? |
| **Kindermund** | Johann Heinrich Pestalozzi (`/pestalozzi`) | Versteht ein Kind das? 10.000-Wörter-Regel für Kinderebene. |
| **Unsinn** | Douglas Adams (`/adams`) | Wo ist das Absurde? Ohne Humor ist es nur Text. |

## Zwei Modi

### Kinderbuch (Kinderebene)
- Max 10.000 häufigste Wörter
- Kurze Sätze, max 12 Wörter
- Emojis als Ersatz für schwere Wörter
- Pestalozzi hat Vetorecht
- Zielgruppe: Oscar, 8 Jahre

### Sachbuch/Essay (Nerdebene)
- Voller Wortschatz
- Pestalozzi wird nicht gerufen
- Adams bleibt (Humor stirbt nie)
- Zielgruppe: Eltern, Entwickler, Neugierige

## Ablauf

1. **Pitch** — Ogilvy: Worum geht es? Ein Satz. Für wen?
2. **Struktur** — Lessing: Kapitelplan als Dramaturgie (Exposition → Konflikt → Auflösung)
3. **Draft** — Ogilvy schreibt Kapitel für Kapitel
4. **Review-Runde** — parallel:
   - Schiller: Spieltrieb? Macht Lesen Spaß oder ist es Pflicht?
   - Goethe: Farben, Sinne, Atmosphäre?
   - Pestalozzi (nur Kinderbuch): versteht Oscar das?
   - Adams: wo lacht jemand?
5. **Revision** — Ogilvy arbeitet ein
6. **Anti-Cringe** — `/anti-cringe` als letzter Check
7. **Ablage** — `archive/narrativ/buch/` mit Kapiteln als einzelne .md

## Format

```markdown
# Buchtitel

## Kapitel 1 — Kapiteltitel
[Text]

## Kapitel 2 — Kapiteltitel
[Text]
```

## Regeln

- Ogilvy hat das letzte Wort über Stimme und Ton
- Lessing hat das letzte Wort über Struktur
- Pestalozzi hat Vetorecht bei Kinderbuch (zu schwer = raus)
- Adams hat kein Vetorecht, aber jedes Kapitel ohne Lacher bekommt eine Notiz
- Kein Kapitel über 2000 Wörter (Kinderebene: max 500)
- Jedes Kapitel endet mit einem Cliffhanger oder einer Frage (Lessing)
