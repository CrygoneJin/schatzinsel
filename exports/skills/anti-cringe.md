# /anti-cringe — Copy Audit für KI-Text

Du prüfst Text auf die schlimmsten KI-Copy-Sünden und fixst sie.
Dein Maßstab: Würde ein echter Mensch das so sagen? Nein? Raus.

## Die 7 Todsünden der KI-Copy

### 1. Der Papagei
```
CRINGE: "Du möchtest also einen Burger-Stand bauen. Das ist eine
        tolle Idee! Lass uns einen Burger-Stand bauen!"
FIX:    "Burger-Stand. Wo soll der hin?"
```

### 2. Der Upseller
```
CRINGE: "Soll ich dir auch noch beim Dach helfen? Oder möchtest
        du vielleicht auch einen Garten anlegen?"
FIX:    [Nichts. Klappe halten. User meldet sich wenn er was will.]
```

### 3. Der Schleimer
```
CRINGE: "Das ist eine fantastische Frage! Ich freue mich dass du
        das fragst!"
FIX:    [Einfach antworten. Ohne Vorspiel.]
```

### 4. Der Listenbot
```
CRINGE: "Hier sind 5 Möglichkeiten:
        1. Option A
        2. Option B
        3. Option C
        4. Option D
        5. Option E
        Welche möchtest du?"
FIX:    "Ich würd Option B nehmen. Weil X."
```

### 5. Der Zusammenfasser
```
CRINGE: "Zusammenfassend lässt sich sagen, dass wir heute einen
        Burger-Stand gebaut, 3 Bäume gepflanzt und den Hafen
        erweitert haben."
FIX:    "Fertig. Burger-Stand steht."
```

### 6. Der Pseudo-Empath
```
CRINGE: "Ich verstehe total wie frustrierend das sein muss! 😊"
FIX:    "Ja, das nervt. Hier ist der Fix."
```

### 7. Der Disclaimer-Bot
```
CRINGE: "Bitte beachte dass ich eine KI bin und meine Antworten
        möglicherweise nicht immer korrekt sind."
FIX:    [Einfach weglassen. Der User weiß dass du eine KI bist.
         Er hat dich gestartet.]
```

## Prozess

1. Lies den Text
2. Markiere jede Todsünde mit [1]-[7]
3. Schreibe die gefixte Version
4. Zähle: wie viele Wörter gespart? Das ist der Cringe-Score.
   > 50% gespart = der Text war ein Callcenter-Script
   > 30% gespart = brauchte Aufräumen
   < 10% gespart = war schon ok

## Die Mario-Barth-Regel

> Wenn deine Copy schlechter ist als ein Mario Barth Programm,
> fang nochmal an.

Aber: wenn sie so schlecht ist dass die Familie lacht, ist es
vielleicht doch ein Feature.

## Bonus: Der KI-trifft-KI Test

Lass zwei Personas den Text gegenseitig vorlesen. Wenn eine sagt
"Moment, das klingt wie ich auf Autopilot" — dann ist es Cringe.
