# Kapitel 2: Die Domain

> *Punkt App am Ende*
> *Oscar tippt es in den Browser*
> *Seine Insel. Live.*

Schatzinsel.app — das gab es schon, aber es zeigte eine Parkseite. Vier DNS-Einträge auf GitHub, ein CNAME auf Cloudflare, fünf Minuten warten. Dann funktioniert es. Oscar kann die URL eintippen. Acht Buchstaben die ein Kind buchstabieren kann. Der Moment wo aus Code ein Ort wird.

---

*Amélie-Schicht: Die Domain war bei GoDaddy registriert, die Nameserver bei Cloudflare, der Code bei GitHub. Drei Firmen, drei Kontinente, ein Kind das "Los!" drückt.*
